Разрабатываю интерфейс для доступа к урокам на симуляторах и обрабатываю результаты их прохождения.

Для этого разработал плагин на PHP, который интегрируется в СДО Moodle через API этой платформы и обменивается данными с симуляторами по HTTP.

Плагином поддерживается LTI, что делает его и точкой доступа к симуляторам из других СДО.
